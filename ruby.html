Baybayin: By character

By character markup is useful for learning the characters in Baybayin.

Markup method #1: Separate ruby block per word

This markup style keeps the ruby-under text on a per-character basis.

ᜀᜃᜓ (A[ɐ]ko) (I) ᜀᜌ᜔ (a[ɐ]y) (am)

Markup method #2: Unified ruby-under

This markup style allows for a unified ruby-under annotation.

ᜁᜐᜅ᜔ ᜉᜋ᜔ᜉᜅ᜕ ( i[ɪ]san͠g pampan͠g ) (a shore)

If we use this method in our “I am” example (see: Markup method #1), we can make it better.

ᜀᜃᜓ (A[ɐ]ko) (I) ᜀᜌ᜔ (a[ɐ]y) (am)

By syllable

By syllable markup is when a student has mastered each Baybayin character. This makes it easier to read as the annotations are syllabical.

ᜀᜃᜓ ᜀᜌ᜔ ᜈᜋᜋᜎᜄᜒ ᜋᜎᜉᜒᜆ᜔ ᜐ ᜁᜐᜅ᜔ ᜉᜋ᜔ᜉᜅ᜕ (a([ɐ])ko a([ɐ])y namamalagi malápit sa i([ɪ])san͠g pampan͠g.) (I am residing near a shore.)

ᜀᜃᜓ ᜀᜌ᜔ (a([ɐ])ko a([ɐ])y) (I am) ᜈᜋᜋᜎᜄᜒ (namamalagi) (residing) ᜋᜎᜉᜒᜆ᜔ (malápit) (near) ᜐ ᜁᜐᜅ᜔ (sa i([ɪ])san͠g) (a) ᜉᜋ᜔ᜉᜅ᜕ (pampan͠g.) (shore.)
